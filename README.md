# 💰 FinTrack Pro

A modern and responsive **Personal Finance Tracker** built using **HTML, CSS, and JavaScript**. Track your income and expenses, visualize cash flow, manage transactions, customize your profile, and monitor your financial health—all in one place.

---

## 🚀 Features

### 🔐 Authentication
- User Registration
- User Login
- LocalStorage Authentication
- Persistent Login State
- Logout Functionality
- Form Validation

### 📊 Dashboard
- Current Balance
- Total Income
- Total Expense
- Total Transactions
- Dynamic Bar Chart (Chart.js)

### 💳 Transaction Management
- Add Transaction
- Edit Transaction
- Delete Transaction
- Search by Description
- Filter by Income/Expense
- Automatic Dashboard Updates
- Real-time Chart Updates

### ⚙️ Settings
- Update Profile Name
- Change Currency
- Live Profile Update
- Live Currency Update
- Data Stored in LocalStorage

### 🎨 UI Features
- Responsive Dashboard
- Dark Mode
- Toast Notifications
- Modern Card UI
- Clean Sidebar Navigation
- Modal-based Transaction Form

### 🗑️ Reset
- Reset All Transactions
- Dashboard Reset
- Chart Reset
- Transaction Table Reset

---
